Lab #2: Coding & Posting a Web Page - The Basics

In this assignment, I converted a Wikipedia article using semantic HTML. The topic, Roko's Basilisk, had come up in another class of mine at the time. I found it interesting, so I chose to use it for this project (apologies for any nightmares this may have caused). Through this assignment, I learned how to use semantic HTML to style a page, as well as separate and markup content to make it more accessible. Unfortunately, at this time we hadn't yet learned about media queries or responsive design, so the website looks a bit bad on mobile devices (you've been warned!).
