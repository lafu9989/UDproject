Lab #3: Tables, Forms, Tabbed Browsing, and Screen Reading

In this assignment, we focused more on specific HTML tagging, as well as HTMl tables and forms. I chose the topic of largest cities in the world because it had a very extensive and large table on the Wikipedia page, so I thought it would be a sufficient challenge for creating a table in HTML. I used the <details> tag to make dropdowns for the definitions provided in the article. I also used Google Icons to create a small form at the bottom of the page where you could (theoretically) look up a city. This website is also not designed to be responsive, so beware of opening it on a mobile device!
